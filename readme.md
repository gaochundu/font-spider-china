



**1、win+R打开cmd命令窗口，全局安装font-spider:**
```shell
npm install font-spider -g
```

**2、查看版本号，查看自己安装是否成功：**
```shell
font-spider  -V
```

**3、运行压缩**
```shell
font-spider index.html

// 如果没有全局安装font-spider
npx font-spider index.html
```
**.font-spider目录下为原来未压缩的字体文件**
